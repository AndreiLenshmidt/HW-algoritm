1. Итоговая работа написана на JavaScript, испольлся только блокнот и Chrome в качестве отладчика. 
Для удобства проверки рекомендую открыть файл index.html => инструменты разработчика => source.
Либо пользоваться любой удобной для вас средой разработки.

2. Реализован класс RedBlackTree с конструктором класса и методом addNode.

addNode принимает рутовую ноду и значение новой ноды, возвращает true если нода успешно добавлена в дерево.

3. Реализован класс Node с конструктором класса и 4 методами балансировки:
- colorSwap(node) - смена цвета нод;
- leftSwap(node) - левый поворот;
- rightSwap(node) - правый поворот;
- rebalance(node) - обратно-рекурсивный метод балансировки нод.
Все методы класса Node статические т.к. работают не с отдельным экземпляром(нодой), а работают с множеством нод, на уровне класса.

4. Реализованны 2 вспомогательные функции:
- showTree() выводит все дерево в JSON-формате;
- checkRootColor (node) следит за тем, чтобы цвет рутовой ноды всегда был черным.
